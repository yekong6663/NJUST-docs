>军事理论由两位老师授课——**王文军老师和冯成老师**。两位老师的授课方式与最终考试差别较大。由于本人只上过王文军老师的课，所以只叙述相关注意事项。

## **王文军老师**
### 课堂纪律
上课时请务必务必将手机等电子设备调节至**静音**！！！老师的考勤方式是，将后门关紧，所有同学按照学号顺序有序排队，依次在书本上盖章。盖完章的同学必须从前门走，不能再回来。也有可能是直接给老师报名字与学号。
### 结课考试
结课作业是一份按照老师**要求的格式**撰写的相关文章，请注意务必务必与同学互相**检查核对格式**！然后就能够收到~~批发的~~80分了。

如果想要获得更高的分数，需要参加考试。考试的内容就是十道填空题，全部都是来自上课的PPT。想要参加考试，需要有**相关的笔记**，考试时可以携带笔记和课本（其实是**开卷考试**）。如果有整理笔记的话，其实难度还是比较小的。
>但是老师上课讲课语速较快，PPT换页也较为迅速，所以最好先拍照再课后整理笔记。
